# 德州扑克游戏 - 需求分析文档

## 一、项目概述

**项目名称**: 德州扑克 Web 游戏
**技术栈**:
- 前端: JavaScript/TypeScript + HTML/CSS
- 后端: Node.js + Express + Socket.io
- 数据库: MongoDB / SQLite
**项目位置**: `poker-game/`

**游戏模式**:
- 单机模式: 与 AI 对手对战
- 多人模式: 网络实时对战（2-6 人）

---

## 二、游戏规则需求

### 2.1 基本规则
- **标准 52 张扑克牌**（不含大小王）
- **2-10 名玩家**（建议 2-6 人局）
- **每人发 2 张底牌**
- **公共牌 5 张**：翻牌 3 张 + 转牌 1 张 + 河牌 1 张

### 2.2 下注轮次
| 轮次 | 说明 | 可选操作 |
|------|------|----------|
| 1. 翻牌前 | 发放底牌后 | 跟注/加注/弃牌/全下 |
| 2. 翻牌 | 发放 3 张公共牌 | 跟注/加注/弃牌/全下 |
| 3. 转牌 | 发放第 4 张公共牌 | 跟注/加注/弃牌/全下 |
| 4. 河牌 | 发放第 5 张公共牌 | 跟注/加注/弃牌/全下 |
| 5. 摊牌 | 剩余玩家亮牌比大小 | - |

### 2.3 牌型排名（从大到小）
1. **皇家同花顺** (Royal Flush)
2. **同花顺** (Straight Flush)
3. **四条** (Four of a Kind)
4. **葫芦** (Full House)
5. **同花** (Flush)
6. **顺子** (Straight)
7. **三条** (Three of a Kind)
8. **两对** (Two Pair)
9. **一对** (One Pair)
10. **高牌** (High Card)

### 4.2 核心类设计

#### Deck（牌堆）
```javascript
class Deck {
  cards: Card[]
  shuffle(): void
  deal(): Card
  reset(): void
}
```

#### Card（扑克牌）
```javascript
class Card {
  suit: 'hearts' | 'diamonds' | 'clubs' | 'spades'
  rank: 2-14 (11=J, 12=Q, 13=K, 14=A)
  toString(): string
}
```

#### HandEvaluator（牌型评估器）
```javascript
class HandEvaluator {
  evaluate(holeCards: Card[], communityCards: Card[]): HandResult
  getHandRank(cards: Card[]): number
  compareHands(hand1: HandResult, hand2: HandResult): number
}
```

#### Game（游戏控制器）
```javascript
class Game {
  players: Player[]
  deck: Deck
  communityCards: Card[]
  pot: number
  currentBet: number
  phase: 'preflop' | 'flop' | 'turn' | 'river' | 'showdown'
  startNewRound(): void
  nextPhase(): void
  handleAction(player: Player, action: Action): void
}
```

## 十、安全性考虑

1. **认证安全**
   - 密码 bcrypt 加密存储
   - JWT Token 过期机制
   - 防止 Token 劫持

2. **游戏安全**
   - 服务端验证所有游戏操作
   - 防止客户端作弊
   - 关键逻辑在服务端执行

3. **通信安全**
   - 使用 WSS (WebSocket Secure)
   - HTTPS 加密传输
   - 输入验证与过滤

4. **数据安全**
   - SQL/NoSQL 注入防护
   - XSS 攻击防护
   - CORS 配置
